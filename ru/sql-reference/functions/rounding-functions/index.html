    –≠—Ç–æ—Ç –º–µ—Ç–æ–¥ –æ–∫—Ä—É–≥–ª–µ–Ω–∏—è, –∏—Å–ø–æ–ª—å–∑—É–µ–º—ã–π –ø–æ —É–º–æ–ª—á–∞–Ω–∏—é –¥–ª—è —á–∏—Å–µ–ª —Å –ø–ª–∞–≤–∞—é—â–µ–π –∑–∞–ø—è—Ç–æ–π, –æ–ø—Ä–µ–¥–µ–ª—ë–Ω –≤ —Å—Ç–∞–Ω–¥–∞—Ä—Ç–µ [IEEE 754](https://en.wikipedia.org/wiki/IEEE_754#Roundings_to_nearest). –§—É–Ω–∫—Ü–∏—è [round](#rounding_functions-round) —Ç–∞–∫–∂–µ –æ–∫—Ä—É–≥–ª—è–µ—Ç —á–∏—Å–ª–∞ —Å –ø–ª–∞–≤–∞—é—â–µ–π –∑–∞–ø—è—Ç–æ–π –ø–æ —ç—Ç–æ–º—É –º–µ—Ç–æ–¥—É. –§—É–Ω–∫—Ü–∏—è `roundBankers` –æ–∫—Ä—É–≥–ª—è–µ—Ç –Ω–µ —Ç–æ–ª—å–∫–æ —á–∏—Å–ª–∞ —Å –ø–ª–∞–≤–∞—é—â–µ–π –∑–∞–ø—è—Ç–æ–π, –Ω–æ –∏ —Ü–µ–ª—ã–µ —á–∏—Å–ª–∞ –º–µ—Ç–æ–¥–æ–º –±–∞–Ω–∫–æ–≤—Å–∫–æ–≥–æ –æ–∫—Ä—É–≥–ª–µ–Ω–∏—è, –Ω–∞–ø—Ä–∏–º–µ—Ä, `roundBankers(45, -1) = 40`.

  • –í –¥—Ä—É–≥–∏—Ö —Å–ª—É—á–∞—è—Ö —Ñ—É–Ω–∫—Ü–∏—è –æ–∫—Ä—É–≥–ª—è–µ—Ç –∫ –±–ª–∏–∂–∞–π—à–µ–º—É —Ü–µ–ª–æ–º—É.

–ë–∞–Ω–∫–æ–≤—Å–∫–æ–µ –æ–∫—Ä—É–≥–ª–µ–Ω–∏–µ –ø–æ–∑–≤–æ–ª—è–µ—Ç —É–º–µ–Ω—å—à–∏—Ç—å –≤–ª–∏—è–Ω–∏–µ –æ–∫—Ä—É–≥–ª–µ–Ω–∏—è —á–∏—Å–µ–ª –Ω–∞ —Ä–µ–∑—É–ª—å—Ç–∞—Ç —Å—É–º–º–∏—Ä–æ–≤–∞–Ω–∏—è –∏–ª–∏ –≤—ã—á–∏—Ç–∞–Ω–∏—è —ç—Ç–∏—Ö —á–∏—Å–µ–ª.

  • –û–∫—Ä—É–≥–ª–µ–Ω–∏–µ –¥–æ –±–ª–∏–∂–∞–π—à–µ–≥–æ —Ü–µ–ª–æ–≥–æ: 2 + 3 + 4 + 5 = 14.
