    -   `tag_list` - –ø—Ä–∞–≤–∏–ª–æ –¥–ª—è —Ç–µ–≥–≥–∏—Ä–æ–≤–∞–Ω–Ω—ã—Ö –º–µ—Ç—Ä–∏–∫, –ø—Ä–æ—Å—Ç–æ–π DSL –¥–ª—è —É–ø—Ä–æ—â–µ–Ω–∏—è –∑–∞–¥–∞–Ω–∏—è —Ä–µ–≥—É–ª—è—Ä–Ω–æ–≥–æ –≤—ã—Ä–∞–∂–µ–Ω–∏—è –≤ —Ñ–æ—Ä–º–∞—Ç–µ —Ç–µ–≥–æ–≤ graphite `someName;tag1=value1;tag2=value2`, `someName` –∏–ª–∏ `tag1=value1;tag2=value2`. –ü–æ–ª–µ `regexp` —Ç—Ä–∞–Ω—Å–ª–∏—Ä—É–µ—Ç—Å—è –≤ –ø—Ä–∞–≤–∏–ª–æ `tagged`. C–æ—Ä—Ç–∏—Ä–æ–≤–∞—Ç—å –ø–æ –∏–º–µ–Ω–∞–º —Ç–µ–≥–æ–≤ –Ω–µ –æ–±—è–∑–∞—Ç–µ–ª—å–Ω–æ, –æ–Ω–æ –æ—Ç—Å–æ—Ä—Ç–∏—Ä—É–µ—Ç—Å—è –∞–≤—Ç–æ–º–∞—Ç–∏—á–µ—Å–∫–∏. –ó–Ω–∞—á–µ–Ω–∏–µ —Ç–µ–≥–∞ (–Ω–æ –Ω–µ –∏–º—è) –º–æ–∂–µ—Ç –±—ã—Ç—å —Ä–µ–≥—É–ª—è—Ä–Ω—ã–º –≤—ã—Ä–∞–∂–µ–Ω–∏–µ–º (–Ω–∞–ø—Ä–∏–º–µ—Ä `env=(dev|staging)`).
  • regexp ‚Äì —à–∞–±–ª–æ–Ω –∏–º–µ–Ω–∏ –º–µ—Ç—Ä–∏–∫–∏ (—Ä–µ–≥—É–ª—è—Ä–Ω–æ–µ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –∏–ª–∏ DSL).
  • age ‚Äì –º–∏–Ω–∏–º–∞–ª—å–Ω—ã–π –≤–æ–∑—Ä–∞—Å—Ç –¥–∞–Ω–Ω—ã—Ö –≤ —Å–µ–∫—É–Ω–¥–∞—Ö.
  • precision ‚Äì —Ç–æ—á–Ω–æ—Å—Ç—å –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –≤–æ–∑—Ä–∞—Å—Ç–∞ –¥–∞–Ω–Ω—ã—Ö –≤ —Å–µ–∫—É–Ω–¥–∞—Ö. –î–æ–ª–∂–µ–Ω –±—ã—Ç—å –¥–µ–ª–∏—Ç–µ–ª–µ–º –¥–ª—è 86400 (–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —Å–µ–∫—É–Ω–¥ –≤ —Å—É—Ç–∫–∞—Ö).
  • function ‚Äì –∏–º—è –∞–≥—Ä–µ–≥–∏—Ä—É—é—â–µ–π —Ñ—É–Ω–∫—Ü–∏–∏, –∫–æ—Ç–æ—Ä—É—é —Å–ª–µ–¥—É–µ—Ç –ø—Ä–∏–º–µ–Ω–∏—Ç—å –∫ –¥–∞–Ω–Ω—ã–º, —á–µ–π –≤–æ–∑—Ä–∞—Å—Ç –æ–∫–∞–∑–∞–ª—Å—è –≤ –∏–Ω—Ç–µ—Ä–≤–∞–ª–µ [age, age + precision]. –î–æ–ø—É—Å—Ç–∏–º—ã–µ —Ñ—É–Ω–∫—Ü–∏–∏: min/max/any/avg. Avg –≤—ã—á–∏—Å–ª—è–µ—Ç—Å—è –Ω–µ—Ç–æ—á–Ω–æ, –∫–∞–∫ —Å—Ä–µ–¥–Ω–µ–µ –æ—Ç —Å—Ä–µ–¥–Ω–∏—Ö.
